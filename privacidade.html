Política de Privacidade

Esta política descreve como tratamos os dados coletados pelo Gerador de Conteúdo IA.

1. Coleta de Dados

Podemos coletar informações fornecidas pelo usuário, como textos inseridos no sistema.

2. Uso das Informações

Os dados são usados apenas para melhorar a experiência do usuário e não são compartilhados.

3. Segurança

Empregamos medidas técnicas para proteger suas informações contra acesso não autorizado.
